# Download instructions
- Download the latest zip file.
- Extract it, open the folder, and run the executeable file. 
- If a SmartScreen warning shows click "More Info" and "Run Anyway".

## Controls 
- WASD to pan up/down and right/left.
- Scroll wheel to zoom in and out.
- Use the number keys 1 through 0 to increase the resolution (1 is the lowest).
- Additional higher resolutions can be accessed by Z through V.
- Press space to enter/exit escape path mode. 
    - This mode shows each points path based on your cursors position.
    - Left click to freeze/unfreeze the path.
